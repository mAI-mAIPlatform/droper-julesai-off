# Titre du jeu : Droper

**Genre** : Battle Royale / Action / Stratégie / Multi-Mode / E-sport / Futuriste
**Style graphique** : Mélange cartoon réaliste + Liquid Glass + futuriste néon, couleurs vives, effets lumineux et particules dynamiques, textures détaillées 4K, HDR, Ray Tracing
**Plateformes** : PC, Xbox Series X/S, PlayStation, iOS, Android
**Gameplay clé** : Battle Royale avec loot, combat tactique, objets interactifs et gadgets, mobilité verticale, compétences uniques, véhicules dynamiques, environnement destructible. Aucune construction classique, remplacée par des objets tactiques : tentes, bulles, mini-boucliers, trampolines, drones, pièges, clones, portails, boosts, tous temporaires et destructibles.

## 2️⃣ Objets tactiques & gameplay (60+)

* **Tentes / abris** → couverture temporaire contre les tirs, destructibles, peuvent se déplacer ou se transformer en points stratégiques.
* **Bulles protectrices** → zones invincibles 5-10 secondes, parfait pour heal ou revival.
* **Mini boucliers / barrières énergétiques** → plaques verticales ou horizontales, temporaires, multi-usage.
* **Trampolines / plateformes flottantes** → prendre hauteur, combo vertical + mobilité rapide.
* **Tourelles / drones automatiques** → tir automatique, défense zone ou distraction.
* **Pièges dynamiques** → mines explosives, champs ralentissants, pièges à grappin, gadgets qui perturbent ennemi.
* **Portails / téléporteurs** → déplacements rapides ou pour bloquer l’ennemi, cooldown variable.
* **Clone / hologramme** → distraction, tromperie, illusions tactiques.
* **Boosts de terrain** → zones qui augmentent vitesse, saut, régénération ou furtivité.
* **Objets interactifs naturels** → rochers, caisses, arbres à déplacer pour créer couverture temporaire.
* **Tuiles énergétiques** → permettent rebondir, propulser ou ralentir.
* **Zones de camouflage temporaires** → invisibilité ou camouflage pour 5-10 sec.
* **Champs gravitationnels** → modifie trajectoire saut ou projectile.
* **Mines de recul / d’attraction** → repousser ou attirer ennemis stratégiquement.
* **Cages / filets** → immobilisation temporaire.
* **Systèmes de soins portables** → objets deployables pour heal zone.
* **Zones de dégâts au sol** → flammes, acide, glace, poison pour contrôle d’espace.
* **Projecteurs / zones lumineuses** → aveuglement tactique.
* **Barrières directionnelles** → bloque certains tirs mais laisse passer alliés.
* **Objets interactifs combinables** → drone + bouclier = turret mobile, trampoline + bulles = rebond + protection.
